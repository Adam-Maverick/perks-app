## PWA Criteria Checklist (NFR1 & NFR2)

### Installability
- [ ] Web App Manifest exists
- [ ] Service Worker registered
- [ ] HTTPS enabled
- [ ] Maskable icon exists

### Offline Capability
- [ ] Responds with 200 when offline (start_url)
- [ ] Custom offline page (optional but recommended)

### Performance (3G)
- [ ] First Contentful Paint (FCP) < 2.0s
- [ ] Largest Contentful Paint (LCP) < 2.5s

## Audit Instructions
1. Open the deployed Vercel URL in Chrome Incognito.
2. Open DevTools (F12) -> Lighthouse tab.
3. Select "Mobile" device.
4. Check "PWA" and "Performance" categories.
5. Click "Analyze page load".
6. Record scores above.
